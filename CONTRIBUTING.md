# Contributing

Contributions to this project are welcome through GitHub issues and pull
requests. Please open an issue to discuss substantive changes before
submitting a pull request, so that scope and approach can be aligned in
advance.

terms, and that the contribution is your original work or is otherwise
properly attributed.
